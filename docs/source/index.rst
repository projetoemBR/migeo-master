MIGEO
======


Modelagem e Inversão Geofísica Eletromagnética
==============================================

MIGEO é um pacote python de modelagem e inversão geofísica eletromagnética por volumes finitos baseada em códigos open-source. Para instalar: ::

    $ pip install migeo
